# Specification

## Summary
**Goal:** Fix channel creation functionality and add comprehensive server discovery and joining features with enhanced UI elements.

**Planned changes:**
- Fix the actor.createChannel error by ensuring the method is properly defined and exported in the backend
- Add a server browser view displaying all public servers with name, member count, and description
- Enhance the join server dialog to support both invite codes and direct selection from the browse list
- Add backend method to list all public servers with metadata
- Improve message composer with visual feedback for mentions, better attachment previews, and enhanced send button states
- Add online status indicators for users in members list and message timeline
- Enhance server cards with icons, member count badges, and online member indicators

**User-visible outcome:** Users can successfully create channels without errors, browse and join public servers through multiple methods, see who is online, and enjoy an improved messaging experience with better visual feedback.
